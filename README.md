# stockReminder

Program is designed to text the stored phone number the prices from their
watchlist.

To run:
  -create a file .env
  -put the following items in the file using Twilio account
    SID =
    TOKEN =
    MY_PHONE =
    COMP_PHONE =

  -update watchlist.txt with stock acronyms
  -enter command: python3 main.py
